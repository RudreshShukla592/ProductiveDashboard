# Productive Dashboard

A modern productivity dashboard built using **HTML**, **CSS**, and **JavaScript**, with **Local Storage** integration to help users stay organized, track daily activities, and boost productivity.

## 🔗 Live Demo

* **Website:** https://rs-productivedashboard.netlify.app/

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* Local Storage
* Git & GitHub
* Netlify

## ✨ Features

* Dynamic Dashboard Interface
* Task & Productivity Widgets
* Light & Dark Theme
* Persistent Data with Local Storage
* Responsive Design
* Clean and Modern UI
* Interactive User Experience

## 🎯 Purpose

This project was created to strengthen my frontend development skills by building a productivity dashboard that combines functionality with a clean, modern user interface. It focuses on creating an interactive user experience while managing data efficiently using the browser's Local Storage.

## 🚀 Key Learning Outcomes

* Building an interactive dashboard interface
* Managing application state with Vanilla JavaScript
* Persisting user data using Local Storage
* Designing responsive and user-friendly layouts
* Creating reusable and maintainable frontend components
